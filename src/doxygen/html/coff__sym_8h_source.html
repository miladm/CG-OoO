00063 /*
00064  * This file contains the definition of the Third Eye Symbol Table.
00065  *
00066  * Symbols are assumed to be in 'encounter order' - i.e. the order that
00067  * the things they represent were encountered by the compiler/assembler/loader.
00068  * EXCEPT for globals!  These are assumed to be bunched together,
00069  * probably right after the last 'normal' symbol.  Globals ARE sorted
00070  * in ascending order.
00071  *
00072  * -----------------------------------------------------------------------
00073  * A brief word about Third Eye naming/use conventions:
00074  *
00075  * All arrays and index's are 0 based.
00076  * All "ifooMax" values are the highest legal value PLUS ONE. This makes
00077  * them good for allocating arrays, etc. All checks are "ifoo < ifooMax".
00078  *
00079  * "isym"       Index into the SYMbol table.
00080  * "ipd"        Index into the Procedure Descriptor array.
00081  * "ifd"        Index into the File Descriptor array.
00082  * "iss"        Index into String Space.
00083  * "cb"         Count of Bytes.
00084  * "rgPd"       array whose domain is "0..ipdMax-1" and RanGe is PDR.
00085  * "rgFd"       array whose domain is "0..ifdMax-1" and RanGe is FDR.
00086  */
00087
